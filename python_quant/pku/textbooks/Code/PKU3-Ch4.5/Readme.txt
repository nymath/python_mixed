此案例提供数据包括（数据来源：Wind）：
(1) 上证50日数据
(2) 上证50月数据
(3) 上证指数日数据
(4) 上证指数月数据
(5) 十年期国债收益率的日数据
(6) 上证50成分股收盘价数据（2011-2020），见“上证50成分股收盘价”文件夹。该文件夹中的文件以各月最后一个【交易日】来命名。具体存储内容为，例如，文件“2010-12-31.csv”中存储的是2010年12月31日当日的上证50指数成分股在该日之前100个【自然日】以及之后1个【自然月】的收盘价数据。

该案例使用的代码见 ch4 - case - new.py，编程语言为python 3。将python工作路径设置为该文件夹即可运行。
